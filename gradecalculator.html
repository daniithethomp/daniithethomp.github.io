GradeCalculator

Github Repository

This project is still a work in progress.

A web application that allows users to enter their module assessments and calculate their grade for a specific test, overall module and overall year. Users can set a target that they want to achieve and get a visual cues on how close to their goal they are.

Features

  • Account creation
  • Account management
  • Total grades for a specific year
  • Overall module grades
  • Test grades
  • Support for threshold tests
  • User set targets

Technologies

  • Ruby on Rails
  • HTML
  • CSS
  • JavaScript
  • Object Relational Mapping
  • Devise
  • CRUD Database

The user's create an account using their email

Once the user is logged in, the home page is a display of all the years they have created, and the option to create a new year.

To create a new year a user just has to enter the year number

Once a year has been created, a user can click on it to view the modules they have created for a year or create a new module for that year.

To create a new module, the user must enter a name for the module and the number of credits the module is worth.

The user can then view all the tests for a particular module that they have created, or create a new test for that module.

To create a test, the user must enter a name for the test, the maximum score they could earn for the test, the actual score they earned and the weighting of the test for the module.

The user also has the option to make the test a threshold test. This replaces the max score, earned score and weight inputs with a single pass fail checkbox. If a user has multiple threshold tests for a particular module, they must pass all of them to achieve 40% in the module.

A user can view all the information for tests they have created by clicking on the test. They can then choose to edit or delete the test.

A user can edit their information (email and password), choose to delete their account, and change their target grade.

Target grades are displayed on tests, modules and years. The black line on each bar represents the target grade. If a user is on or above their target grade the bar is green, if they are below by 10% or less then the bar is yellow, and if they are below by more 10% the bar is red.
